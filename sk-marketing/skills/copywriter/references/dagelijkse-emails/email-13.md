TITEL: Doe dit vanaf VANDAAG (over een paar maanden ben je happy)

Voordat je in Claude Code duikt. Voordat je marketing workflows bouwt...

... is er één ding dat je vanaf VANDAAG serieus wilt doen.

En bijna niemand doet dit constructief.

Je wilt een "internet knowledge base" aanleggen.

Basically het swipe file concept. Maar dan next level.

Want dit is wat je AI outputs transformeert van generiek naar waardevol.

Zo werkt het:

Vanaf nu sla je ALLES op dat opvalt online.

Landing pagina's.
YouTube thumbnails.
Branding assets.
Goede copywriting.
Tweets, LinkedIn posts, TikToks, Reels.
Stukjes code.
Emails die je opent (zoals deze, lol).

Alles wat je denken laat "damn, dit is goed" - save it.

Sla het op in mappen. Notion. Obsidian. Maakt niet uit. Als Claude Code er maar bij kan.

En dan... injecteer je deze knowledge base bij ALLES wat je met AI doet.

Als referentie. Als inspiratie. Als voorbeelden van wat WEL en wat NIET werkt.

Waarom dit zo belangrijk is?

Context is alles.

De komende jaren wordt dit alleen maar belangrijker. AI modellen worden krachtiger, maar zonder de juiste context krijg je dezelfde generieke output als iedereen.

Met jouw eigen verzameling van jaren aan "dit vind ik goed" en "dit niet"... krijg je output die JOUW smaak heeft.

Je eigen stijl.

Je eigen denken.

Begin vandaag. Elke dag een paar saves. Over een paar maanden heb je een goudmijn.

Groet Mitch

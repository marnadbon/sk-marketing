Het is een wat hoger risico, maar ook stuk hogere rewards als je een goede NFT set uitkiest. Dit heb ik een aantal keer mogen ervaren, en je 0.05 ETH kan binnen 2 a 3 dagen ineens 2 ETH waard zijn.

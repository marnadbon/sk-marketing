# Bronnen — Copywriting Trainingsmateriaal

Voeg hier je eigen trainingsmateriaal toe over copywriting:
- Cursussamenvattingen
- Boeknotities (bijv. eigen samenvattingen van copywriting boeken)
- Workshop transcripties
- Framework documentatie
- Swipe files

Deze bestanden worden automatisch meegenomen als context bij het schrijven van teksten.

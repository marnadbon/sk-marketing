# X/Twitter Post Templates

15 templates voor X/Twitter. Elke tweet moet het bars-principe volgen: elke zin eindigt met impact.

Onthoud: 280 tekens is de standaard. Elk woord moet zijn plek verdienen.

## A. Enkele Tweets

### 1. One-Liner Insight

**Doel:** Positioneren | **Tekens:** 100-200

Eén zin die een heel idee samenvat. Deelbaar, bookmarkbaar, memorabel.

**Structuur:**
```
[Eén krachtige observatie of waarheid — max 2 zinnen]
```

**Regels:**
- Maximaal 2 zinnen
- Moet op zichzelf werken zonder context
- De laatste woorden moeten een "bar" zijn
- Specifiek > vaag

**Voorbeeld 1:**
```
Je hebt geen 10.000 volgers nodig om €10K per maand te verdienen.

Je hebt 100 mensen nodig die je vertrouwen.
```

**Voorbeeld 2:**
```
Het verschil tussen €3K en €30K per maand is niet meer werken.

Het is beter kiezen wie je helpt.
```

**Voorbeeld 3:**
```
Consistentie is niet elke dag posten.

Consistentie is elke dag beter worden in wat je post.
```

---

### 2. Hot Take / Contrarian

**Doel:** Positioneren | **Tekens:** 150-280

Ga tegen de gangbare mening in. Niet voor provocatie, maar vanuit ervaring.

**Structuur:**
```
[Wat iedereen gelooft — 1 zin]

[Waarom dat niet klopt — 1-2 zinnen]

[Wat wél werkt — 1 zin met bar]
```

**Regels:**
- Onderbouw met ervaring, niet met mening
- De contrarian claim moet specifiek zijn
- Eindig met je alternatief (de bar)

**Voorbeeld 1:**
```
"Post elke dag" is het slechtste advies op X.

Ik groeide sneller toen ik van 7x naar 3x per week ging en de rest van mijn tijd investeerde in replies.

Engagement bouwt sneller dan volume.
```

**Voorbeeld 2:**
```
Funnels zijn niet dood.

Slechte funnels zijn dood.

Het verschil: de meeste mensen bouwen een funnel voor zichzelf, niet voor hun klant.
```

**Voorbeeld 3:**
```
Je hoeft niet "je passie te volgen."

Volg je nieuwsgierigheid. Passie komt nadat je ergens goed in wordt, niet ervoor.
```

### 3. Specifiek Resultaat

**Doel:** Bewijs | **Tekens:** 150-280

Deel een concreet resultaat met context. Geen opschepperij — focus op de les.

**Structuur:**
```
[Het resultaat — specifiek getal + tijdframe]

[Wat je deed — 1-2 zinnen]

[De les — 1 zin met bar]
```

**Regels:**
- Altijd een specifiek getal + tijdframe
- Focus op wat je DEED, niet op het resultaat alleen
- De les is het deel dat mensen bookmarken

**Voorbeeld 1:**
```
Van 200 naar 5.000 volgers in 4 maanden.

Eén thread per week. 10 replies per dag. Geen gimmicks, geen shortcuts.

De reply game is de beste groeistrategie die niemand wil doen.
```

**Voorbeeld 2:**
```
€23K omzet in maart. Zonder ads.

3 LinkedIn posts per week → 1 nieuwsbrief → 1 aanbod.

De funnel is simpeler dan je denkt. De executie is harder.
```

**Voorbeeld 3:**
```
Eerste 1.000 volgers kostten me 8 maanden. De volgende 4.000 kostten 3 maanden.

Het verschil: ik stopte met zenden en begon met reageren.
```

### 4. Reis Momentopname

**Doel:** Nurture | **Tekens:** 150-280

Deel een moment uit je dag/week/reis en koppel het aan een inzicht. Dit is het template voor "vandaag had ik een videograaf" type ideeën.

**Structuur:**
```
[Het moment — wat er gebeurde, specifiek]

[Wat je realiseerde of leerde — 1-2 zinnen]

[De bredere les — 1 zin met bar]
```

**Regels:**
- Begin met het MOMENT, niet met de les
- Specifieke details maken het echt (naam, getal, locatie, tool)
- De les komt als payoff, niet als preek

#### Sub-variatie A: Behind-the-Scenes Moment → Les

Voor momenten uit je werkdag die een inzicht opleveren.

**Voorbeeld:**
```
Vandaag had ik een videograaf over de vloer.

4 uur filmen voor content die 3 maanden meegaat. Kosten: €600. ROI als zelfs 1 klant binnenkomt: 10x.

De duurste content is de content die je niet maakt.
```

#### Sub-variatie B: Investering/Beslissing → Filosofie

Voor momenten waarop je een investering doet of een keuze maakt.

**Voorbeeld:**
```
Net €3.000 overgemaakt voor een mastermind die pas in juni start.

Vorig jaar zou ik dat geld "bewaard" hebben. Nu weet ik: de duurste investering is de investering die je niet doet.

Groei kost geld. Stilstand kost meer.
```

#### Sub-variatie C: Milestone → Reflectie

Voor het bereiken van een mijlpaal, groot of klein.

**Voorbeeld:**
```
Vandaag mijn 100e nieuwsbrief verstuurd.

De eerste had 43 lezers. Deze heeft er 2.400. Geen hack, geen shortcut. Gewoon 100 weken niet stoppen.

De meeste mensen stoppen bij week 12. Daar begint het pas.
```

#### Sub-variatie D: Uitdaging → Oplossing

Voor momenten waarop je een probleem tegenkomt en oplost.

**Voorbeeld:**
```
Mijn lancering flopte vorige week. €0 omzet op dag 1.

Dag 2: salespage herschreven. Dag 3: €4.200.

De eerste versie is nooit de goede. De snelheid waarmee je aanpast bepaalt je succes.
```

#### Sub-variatie E: Menselijk Moment → Connectie

Voor kwetsbare of persoonlijke momenten die verbinden.

**Voorbeeld:**
```
Mijn vriendin vroeg gisteren: "Wanneer ben je eigenlijk klaar met werken?"

Eerlijk antwoord: dat weet ik niet altijd.

Ondernemerschap is geen 9-tot-5. Het is leren wanneer genoeg genoeg is. Daar ben ik nog niet.
```

### 5. Herkenbaarheid

**Doel:** Engagement | **Tekens:** 150-280

Beschrijf een situatie zo herkenbaar dat je doelgroep denkt: "Dit ben ik."

**Structuur:**
```
[De herkenbare situatie — specifiek en visueel]

[De twist of het inzicht — 1 zin]
```

**Regels:**
- Hoe specifieker het moment, hoe herkenbaarder
- Geen moraal nodig — de herkenning IS de waarde
- Werkt het best met "als je ooit..." of een beschrijvend moment

**Voorbeeld 1:**
```
Je om 23:00 nog je salespage herschrijven omdat de headline "niet helemaal lekker zit."

Welkom bij ondernemerschap.
```

**Voorbeeld 2:**
```
Die fase waarin je tegelijk te veel en te weinig klanten hebt.

Te veel om te groeien. Te weinig om iemand aan te nemen.

Elke ondernemer kent dit punt. Niemand praat erover.
```

**Voorbeeld 3:**
```
Iemand vraagt: "Wat doe je voor werk?"

Je probeert het in één zin uit te leggen. Het worden er vijf. Ze knikken beleefd.

Het teken dat je iets bouwt dat nog niet in een hokje past.
```

### 6. Simpele Vergelijking

**Doel:** Positioneren | **Tekens:** 100-250

Maak een complex idee helder door het te vergelijken met iets herkenbaars.

**Structuur:**
```
[X is als Y]

[Waarom — 1-2 zinnen die de vergelijking uitwerken]
```

**Regels:**
- De vergelijking moet onverwacht maar logisch zijn
- Werk het uit — niet alleen de vergelijking maar het WAAROM
- Kort. De kracht zit in de eenvoud.

**Voorbeeld 1:**
```
Content maken is als naar de sportschool gaan.

De eerste 3 maanden zie je geen resultaat. Maand 4 verandert alles.

De meeste mensen stoppen in maand 2.
```

**Voorbeeld 2:**
```
Je funnel is als een restaurantmenu.

Als alles "chef's special" is, kiest niemand iets. Eén signature dish verkoopt beter dan 30 opties.
```

**Voorbeeld 3:**
```
Ondernemen is als koken.

Beginners volgen recepten. Gevorderden improviseren. Experts maken hun eigen recepten.

De meeste mensen zoeken eeuwig naar het perfecte recept. Begin met koken.
```

### 7. Directe Actie

**Doel:** Converteren | **Tekens:** 150-280

Spoor je publiek aan tot een specifieke actie. Geen vage motivatie — concreet en uitvoerbaar.

**Structuur:**
```
[De situatie waar je lezer in zit — 1 zin]

[De actie die ze moeten nemen — direct en specifiek]

[Waarom nu — 1 zin met urgentie of bar]
```

**Regels:**
- De actie moet vandaag uitvoerbaar zijn
- Geen "overweeg eens..." maar "doe dit"
- Urgentie zonder hype

**Voorbeeld 1:**
```
Als je al 3 maanden twijfelt over die cursus lanceren:

Open een Google Doc. Schrijf 5 lessen die je klanten steeds opnieuw vertelt. Dat is je curriculum.

Het perfecte moment bestaat niet. Dit moment wel.
```

**Voorbeeld 2:**
```
Stop met scrollen. Open je email tool. Stuur je lijst een mail.

Het maakt niet uit wat. Een tip. Een verhaal. Een vraag.

De meeste lijsten sterven niet door slechte content. Ze sterven door stilte.
```

**Voorbeeld 3:**
```
Je hebt deze week nog niet gepost.

Neem je telefoon. Open X. Schrijf één zin over wat je vandaag leerde.

Post hem. Niet morgen. Nu.
```

### 8. Framework Thread

**Doel:** Aantrekken | **Tweets:** 5-8

Leg een methode of systeem uit in stappen. Hoge bookmark-rate.

**Structuur:**
```
TWEET 1 (HOOK):
[Belofte — wat de lezer gaat leren + waarom het werkt]
"Mijn [naam]-systeem in [X] stappen:"

TWEET 2 (CONTEXT):
[Waarom dit framework bestaat — welk probleem het oplost]
[Jouw geloofwaardigheid — kort]

TWEET 3-6 (STAPPEN):
[X/] [Stap naam]
[Uitleg in 1-2 zinnen]
[Voorbeeld of specifiek detail]

TWEET 7 (SAMENVATTING):
TL;DR:
[Bullet points van alle stappen]

TWEET 8 (CTA):
[Retweet tweet 1 als dit waardevol was]
[Volg me voor meer [onderwerp]]
```

**Voorbeeld Hook:**
```
Mijn Content Waterfall systeem in 5 stappen.

Hiermee maak ik van 1 tweet 26+ stukken content over alle platforms.

Een thread 🧵
```

---

### 9. Reis/Story Thread

**Doel:** Nurture | **Tweets:** 5-10

Vertel een persoonlijk verhaal van A naar B. Origin stories, doorbraken, of mislukkingen.

**Structuur:**
```
TWEET 1 (HOOK):
[Het resultaat of de cliffhanger — specifiek]

TWEET 2 (DE SITUATIE):
[Waar was je? Wat was het probleem?]

TWEET 3-7 (HET VERHAAL):
[Chronologisch: wat gebeurde er?]
[Specifieke momenten, beslissingen, getallen]
[Het keerpunt]

TWEET 8-9 (DE LES):
[Wat leerde je?]
[Wat zou je anders doen?]

TWEET 10 (CTA):
[Herkenbaar? Retweet tweet 1]
[Volg voor meer van dit soort verhalen]
```

**Voorbeeld Hook:**
```
In 2023 had ik €0 online omzet.

12 maanden later: €147K. Geen ads. Geen volgers bij de start. Geen netwerk.

Dit is precies wat ik deed. Een thread 🧵
```

---

### 10. Lessen Lijst Thread

**Doel:** Aantrekken | **Tweets:** 7-12

Een genummerde lijst van lessen, tips, of inzichten. Consistent hoge engagement op X.

**Structuur:**
```
TWEET 1 (HOOK):
[X lessen na [specifieke ervaring/periode/bedrag]]

TWEET 2-10 (LESSEN):
[nummer/] [De les — 1 zin]
[Uitleg of voorbeeld — 1-2 zinnen]

TWEET 11 (SAMENVATTING):
Samengevat:
[De 3 belangrijkste lessen als bullet points]

TWEET 12 (CTA):
[Welke les resoneerde het meest?]
[Retweet tweet 1 voor iemand die dit nodig heeft]
```

**Voorbeeld Hook:**
```
7 lessen na €500K omzet die ik wou dat iemand me 5 jaar geleden had verteld:

Een thread 🧵
```

---

### 11. Proof/Case Study Thread

**Doel:** Converteren | **Tweets:** 5-8

Laat een concreet resultaat zien met het proces erachter. Ideaal met screenshots.

**Structuur:**
```
TWEET 1 (HOOK):
[Het resultaat — specifiek, meetbaar, opvallend]

TWEET 2 (CONTEXT):
[De uitgangssituatie — waar begon je/de klant?]

TWEET 3-5 (HET PROCES):
[Stap voor stap wat je deed]
[Specifieke tools, tactieken, keuzes]
[Screenshots als bewijs]

TWEET 6 (HET RESULTAAT):
[Harde cijfers — voor/na vergelijking]

TWEET 7 (DE LES):
[Wat was de sleutel? Wat kunnen anderen hiervan leren?]

TWEET 8 (CTA):
[Wil je weten hoe? DM me "systeem"]
[Of: Retweet voor iemand die dit nodig heeft]
```

**Voorbeeld Hook:**
```
Hoe ik mijn nieuwsbrief van 43 naar 2.400 lezers groeide.

Zonder ads. Zonder samenwerkingen. Met 1 strategie.

Alles wat ik deed 🧵
```

---

### 12. Contrarian Argument Thread

**Doel:** Positioneren | **Tweets:** 5-7

Neem een standpunt in dat tegen de consensus ingaat en onderbouw het volledig.

**Structuur:**
```
TWEET 1 (HOOK):
[De contrarian stelling — provocerend maar eerlijk]

TWEET 2 (WAT IEDEREEN GELOOFT):
[De gangbare mening, eerlijk weergegeven]

TWEET 3 (WAAROM HET NIET KLOPT):
[Jouw argument met specifiek bewijs]

TWEET 4-5 (JE ALTERNATIEF):
[Wat wél werkt + waarom]
[Eigen resultaten als bewijs]

TWEET 6 (DE NUANCE):
[Wanneer de gangbare mening WEL klopt]
[Laat zien dat je eerlijk bent, niet contrair om contrair te zijn]

TWEET 7 (CTA):
[Eens of oneens? Quote-tweet met je perspectief]
```

**Voorbeeld Hook:**
```
Unpopular opinion: je hebt geen content strategie nodig.

Je hebt een posting habit nodig. Strategie komt later.

Hier is waarom 🧵
```

## C. Image/Media Posts

---

### 13. Screenshot + Insight

**Doel:** Bewijs | **Type:** Single image

Deel een screenshot (resultaat, dashboard, DM, review) met een kort inzicht.

**Structuur:**
```
[1-2 zinnen context over de screenshot]

[De les of het inzicht — 1 zin met bar]

[Screenshot als bijlage]
```

**Regels:**
- De screenshot moet zelfstandig leesbaar zijn
- De tekst voegt context toe die de screenshot niet geeft
- Geen gevoelige informatie zichtbaar (blur namen/bedragen als nodig)

**Voorbeeld:**
```
Dit dashboard checkte ik elke ochtend het afgelopen jaar.

12.3M impressies. 8% engagement rate. Alles organisch.

Niet door een hack. Door 365 dagen niet stoppen.

[Screenshot van analytics dashboard]
```

---

### 14. Multi-Image Bewijs

**Doel:** Converteren | **Type:** 2-4 images

Combineer meerdere screenshots of beelden voor een overtuigend bewijs-verhaal.

**Structuur:**
```
[De claim — wat je bewijst]

[Context per afbeelding — kort]

[De les of CTA]

[2-4 afbeeldingen als bijlage]
```

**Regels:**
- Elke afbeelding vertelt een deel van het verhaal
- De volgorde moet logisch zijn (chronologisch of opbouwend)
- De tweet-tekst verbindt de beelden

**Voorbeeld:**
```
Van €0 naar €23K/maand in 8 maanden.

Foto 1: Mijn Stripe van januari — €0
Foto 2: Mijn eerste verkoop — €297
Foto 3: De maand dat het klikte — €8.400
Foto 4: Maart deze week — €23.100

Geen magic. Eén aanbod. Eén platform. Eén systeem.
```

### 15. Before/After

**Doel:** Nurture | **Type:** 2 images

Laat een transformatie zien met twee contrasterende beelden.

**Structuur:**
```
[Tijdframe of context — 1 zin]

Links: [Beschrijving van de 'voor']
Rechts: [Beschrijving van de 'na']

[De les — wat veranderde en waarom]

[2 afbeeldingen: voor en na]
```

**Regels:**
- Het contrast moet visueel duidelijk zijn
- Voeg altijd een tijdframe toe
- De les maakt het relevant voor anderen (niet alleen opscheppen)

**Voorbeeld:**
```
12 maanden verschil.

Links: mijn eerste tweet. 2 likes. Van mijn moeder en mijn vriendin.
Rechts: mijn tweet van gisteren. 847 likes. 12.000 impressies.

Het verschil: niet beter schrijven. Beter luisteren naar wat mijn publiek wilde lezen.
```

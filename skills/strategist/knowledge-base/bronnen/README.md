# Bronnenmateriaal — Eigen Toevoegingen

> Deze map is bedoeld voor je eigen bronmateriaal: boektranscripties, cursusnotities, workshopsamenvattingen en andere referenties die je wilt toevoegen aan de kennisbank van de Strategist.

## Hoe Bronmateriaal Toe te Voegen

### Stap 1: Bereid het materiaal voor

Converteer je bronmateriaal naar Markdown (.md) formaat. Dit kan zijn:

- **Boeken**: Notities, samenvattingen of volledige transcripties van marketingboeken
- **Cursustranscripties**: Transcripties van video-cursussen of online trainingen
- **Workshopnotities**: Aantekeningen van workshops, masterclasses of conferenties
- **Podcasts**: Transcripties van relevante podcast-afleveringen
- **Eigen analyses**: Je eigen samenvattingen en analyses van bronmateriaal

### Stap 2: Bestandsnaamgeving

Gebruik duidelijke, beschrijvende bestandsnamen in kebab-case:

```
value-based-pricing-samenvatting.md
funnel-architectuur-cursusnotities.md
personal-branding-workshop-2024.md
email-marketing-boeknotities.md
```

### Stap 3: Structuur binnen het bestand

Elke bron-bestand zou idealiter deze structuur volgen:

```markdown
# [Titel van het Bronmateriaal]

> Bron: [Waar het vandaan komt]
> Type: [Boek / Cursus / Workshop / Podcast / Eigen analyse]
> Datum: [Wanneer gecreëerd of geconsumeerd]

## Kernpunten

- [Belangrijkste inzicht 1]
- [Belangrijkste inzicht 2]
- [etc.]

## Gedetailleerde Notities

[Je volledige notities hier...]

## Toepasbare Frameworks

[Frameworks of methodes die je kunt toepassen...]
```

### Stap 4: Update de index (optioneel)

Als je een belangrijk bronbestand toevoegt, kun je het vermelden in `../index.md` onder de relevante categorie, zodat de Strategist agent het makkelijk kan vinden.

## Tips

- **Gebruik koppen en structuur**: Dit maakt het makkelijker voor de agent om specifieke secties te vinden met Grep
- **Bewaar exacte citaten**: Als een bron een krachtige formulering heeft, bewaar het als citaat
- **Voeg je eigen interpretatie toe**: Leg uit WAAROM een concept belangrijk is voor jouw situatie
- **Koppel aan bestaande frameworks**: Verwijs naar frameworks uit `../frameworks.md` waar relevant
- **Maak het doorzoekbaar**: Gebruik duidelijke koppen en trefwoorden

## Voorbeeld Bestandsstructuur

```
bronnen/
  value-pricing-notities.md
  funnel-masterclass-transcriptie.md
  personal-brand-workshop.md
  email-copywriting-cursus.md
  marktonderzoek-methoden.md
```

## Belangrijk

- Plaats alleen materiaal waar je de rechten toe hebt (eigen notities, samenvattingen, aantekeningen)
- De Strategist agent gebruikt dit materiaal als aanvullende context bij het geven van advies
- Hoe meer relevant bronmateriaal je toevoegt, hoe dieper en specifieker het advies wordt
